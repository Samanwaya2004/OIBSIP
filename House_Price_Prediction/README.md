# House Price Prediction using Linear Regression

## Objective
The objective of this project is to build a Linear Regression model to predict house prices based on various features such as area, bedrooms, bathrooms, stories, parking, and furnishing status.

## Dataset
- Housing Dataset
- Total Records: 545

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Imported the required libraries.
2. Loaded the housing dataset.
3. Explored and cleaned the data.
4. Converted categorical variables into numerical values.
5. Split the dataset into training and testing sets.
6. Trained a Linear Regression model.
7. Predicted house prices.
8. Evaluated the model using MAE, MSE, and R² Score.
9. Visualized Actual vs Predicted house prices.

## Results

- Mean Absolute Error (MAE): **970043.403**
- Mean Squared Error (MSE): **1754318687330.6628**
- R² Score: **0.6529**

## Conclusion

The Linear Regression model achieved an **R² Score of 0.6529**, indicating that it explains approximately **65.29% of the variation** in house prices. The model provides reasonably accurate predictions for the given dataset.
